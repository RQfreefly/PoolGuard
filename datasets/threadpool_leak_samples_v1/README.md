# sample_dataset_v1

这是用于 Java 线程池泄露扫描 Agent 的合成标注样本集（v1）。

## 目录结构

- `java/`: Java 样本代码（正例 + 反例）
- `labels.jsonl`: 样本标注（每行一个 JSON 对象）

## 设计目标

1. 优先覆盖 MVP 规则：`PG001`、`PG002`、`PG003`、`PG006`。
2. 同时补充少量扩展规则：`PG008`、`PG009`、`PG010`。
3. 每个样本尽量保持小而清晰，便于规则调试与回归测试。

## 样本规模（v1）

1. 总样本数：`50`
2. 正例（应报问题）：`25`，ID 范围：`S001-S025`
3. 反例（不应报问题）：`25`，ID 范围：`S101-S125`

## 标注字段说明

- `sample_id`: 样本 ID
- `label`: `positive`（应报问题）或 `negative`（不应报问题）
- `primary_rule`: 主规则 ID，反例为 `NONE`
- `file`: 样本文件相对路径（相对于本目录）
- `expected_findings`: 期望命中的问题列表
  - `rule_id`: 规则 ID
  - `severity`: 期望严重程度
  - `reason`: 期望原因摘要
- `notes`: 说明

## 说明

1. 该样本集为合成数据，可用于工具冷启动、规则回归和提示词调试。
2. 后续建议用真实历史案例逐步替换或补充，以提高评估可信度。
